# up250927-pyton
Este repositorio tiene la finalidad de almacenar mis trabajos realizados en la asignatura de metrología de la programación en la carrera de ingeniería mecatrónica de la Universidad Politécnica de Aguascalientes

El programa fue diseñado para ser facil de usar, incluso para personas que apenas estan aprendiendo a programr. Se ejecuta desde la terminal y guarda la informacion en un archivo local.
Proposito

El propósito del proyecto es aplicar los conocimientos básicos de programación en Python, como:

Uso de Variables 

Estructuras de Control 

Funciones

Manejo de Archivos

Lista y Ciclos

Alcance:

El Sistema Permite:

Agregar Tareas Nuevas 

Mostrar lista de Tareas 

Marcar Tareas como Completadas 

Eliminar Tareas 

Guardar la informacion para usarla

¿Para que sirve el Repositorio?

Este repositorio sirve para:

Almacenar el codigo del Proyecto

Compartir el Programa con otras personas 

Ejecutar el sistema en cualquier computadora

Modificar o mejorar el codigo en el futuro

 Instalación de Git
Git es un programa que permite descargar y manejar repositorios.

Paso 1
Ir a la página oficial:
https://git-scm.com

Paso 2
Descargar la versión para tu sistema operativo (Windows, Mac o Linux).

Paso 3
Ejecutar el instalador y dejar las opciones por defecto.

Paso 4
Verificar que se instaló correctamente abriendo la terminal o símbolo del sistema y escribiendo:

git --version
Si aparece un número de versión, Git está listo.

2️Instalación de Python
Python es el lenguaje en el que está hecho el proyecto.

Paso 1
Ir a la página oficial:
https://www.python.org

Paso 2
Descargar la versión más reciente de Python 3.

Paso 3
IMPORTANTE: Durante la instalación marcar la casilla
Add Python to PATH

Paso 4
Verificar la instalación abriendo la terminal y escribiendo:

python --version
Instalación de Visual Studio Code
VS Code es el editor donde se puede ver y modificar el código.

Paso 1
Ir a:
https://code.visualstudio.com

Paso 2
Descargar e instalar.

Paso 3
Abrir VS Code y en la sección de extensiones instalar:
Python (extensión oficial de Microsoft)

Clonar el repositorio
Una vez instalado todo:

Paso 1
Abrir la terminal

Paso 2
Escribir el siguiente comando (cambia el enlace por el de tu repositorio):

git clone https://github.com/usuario/nombre-del-repositorio.git
Paso 3
Entrar a la carpeta del proyecto:

cd nombre-del-repositorio
Ejecutar el programa
Dentro de la carpeta del proyecto, ejecutar:

python main.py
(Cambia main.py por el nombre real de tu archivo principal si es diferente)

El programa se abrirá en la terminal y podrás empezar a usarlo.
